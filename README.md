# AI Math Study

이 저장소는 인공지능 모델을 이해하고 구현하기 위해 필요한  
수학적 기초 개념을 정리한 학습 기록입니다.

단순한 공식 암기보다는,  
**각 수학 개념이 AI 모델에서 어떤 역할을 하는지**를 이해하는 것을 목표로 합니다.

---

## 🎯 Study Goal
- 인공지능/머신러닝 모델의 수학적 원리 이해
- 수식이 실제 코드와 알고리즘에서 어떻게 사용되는지 연결
- 논문 및 모델 구조를 수학적으로 해석할 수 있는 기반 마련

---

## 📚 Covered Topics
- **Linear Algebra**  
  벡터, 행렬, 내적, 고유값 등 모델 연산의 핵심

- **Probability & Statistics**  
  확률변수, 분포, 기댓값, 베이즈 정리 등 불확실성 모델링

- **Calculus**  
  미분, 편미분, 체인룰, 경사 계산

- **Optimization**  
  Gradient Descent, 비용 함수, 수렴 개념

---

## 🧠 Study Method
1. 개념 및 수식 정리
2. AI 모델에서의 사용 맥락 설명
3. 간단한 예제 또는 직관적 해석 추가
4. 필요 시 코드/실험과 연결
