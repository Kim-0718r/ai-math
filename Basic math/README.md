1장 방정식과 부등식

- 변수 : 변하는 수
- 상수 : 고정되어 있는 값

y = 2x + 3

만약 x에다가 1을 넣으면 5가 되고 2를 넣으면 7이 된다. 

뒤에 +3은 상수

y = x + 4

x 에 2를 넣으면 6 3을 넣으면 7이 되어 x는 변수 +4는 상수이다.

항 : 숫자 또는 문자의 곱으로 이루어진 식

3x

상수항 : 숫자만 있는 항

1,2,3

계수 : 상수와 변수로 이루어진 단항식에서 변수와 곱해진 상수

3x 에서 3이 계수

Ex)

2x + 3y -3  

각각 2x는 항 , 3y도 항 -3은  상수항이면서 항

계수는 순차적으로 2, 3, 1 이다.

단항식 : 하나의 항으로 이루어진 식

2x + 3y -3 이 식은 각각 개별적으로 단항식이다.

다항식 : 하나 또는 2개 이상의 항의 합으로 이루어진 식

2x + 3y -3 의 식 전부 다항식이다.

차수 : 문자가 곱해진 횟수

2x^2 는 x를 두번 곱한 것 2x^2의 차수는 2가 된다.

5x^2  + 3x -6y +4 라는 식 가정하면 

1. 다항식 (전체의 소식 자체)
2. +4는 상수항
3. 각 항의 차수는 5x^2는 2 3x는 1  6y는 1  4는 0이된다.

수식 : 변수와 상수를 연산자를 이용하여 표현한 식

연산자란 (+,-,*,/) 등 계산하는 식 

a + 3 에서 a ,3은 피연산자 +는 연산자

y = ax + b > 수식

y = ax + b + d > 수식

등식 : 식에 등호(=)가 있으면 그 식이 맞던 틀리던 등식이라고 한다.

2 + 2 = 4  왼쪽을 좌변 오른쪽을 우변이라고 한다.

2 + 5 = 3 + 4 (등식)

참인 등식 : 9 = 6 + 3

거짓인 등식 : 7 + 2 = 6 - 4

방정식 : x와 같은 미지수가 있어서, 그 미지수에 따라 식이 참이 되기도 하고 거짓이 되기도 하는 것

방정식은 반드시 미지수와 등호가 있어야 된다.

미지수란 : 알고 있지 않는 어떠한 수 

2x + 1 에서 x가 미지수 

일차 방정식 : 차수가 1인 방정식

일차방정식의 풀이는 다음과 같은 순서로 진행된다.

1. 변수는 모두 좌변으로 , 상수는 모두 우변으로 이항한다.
2. 각 변을 정리한다.
3. x의 계수로 양변을 나눈다.

이차 방정식 : 차수가 2인 방정식

ax^2 + bx + c = 0

만약 a =0 이면 차수가 1로 바뀐다. 

a 가 0이 아니여야된다는 조건이 있다. 

항등식 : 미지수에 어떤 수를 대입하더라도 항상 참이 되는 식

2x + 1 = 1 + 2x

| 구분 | 방정식 | 항등식 |
| --- | --- | --- |
| 참이 수식인 조건 | 미지수가 특정한 값을 가질 때만 참인 수식 | 미지수가 어떤 값을 가져도 참인 수식 |
| 좌변과 우변의 조건 | 좌변 ≠ 우변 | 좌변 = 우변  |

연립방정식 : 미지수가 여러 개 포함된 방정식들을 묶어 놓은 것

- 3x + y = 2
- x - 2y = 3

- x + y = 2
- y + z = 5
- z - x = 3

부등식 : 부등호(< , ≤ < > , ≥ )를 사용하여 나타낸 식

- 절대 부등식 : 모든 실수의 값에 대하여 항상 성립하는 부등식
    - x^2 + 5 ≥ 0
- 조건 부등식 : 어떤 실수의 값에 대해서만 성립하는 부등식
    - x  + 2 ≤ 7

부등식의 성질

- 부등식의 양변에 같은 수를 더할 경우
    - 5  + 5> 4 + 5 = 10 > 9
- 부등식의 양변에 같은 수를 뺀 경우
    - 5  - 2 > 4 - 2 = 3 > 2
- 부등식의 양변에 같은 수를 곱할 때,
    - 양수를 곱할 경우
        - 5 * 2> 4 *2 = 10 > 8
    - 음수를 곱할 경우
        - 5 * -2 > 4 * 2 = -10 < -8
    - 양수를 나눌 경우
        - 부등호는 고정
    - 음수를 나눌 경우
        - 부등호의 방향이 바뀜
